# simply-vue

[![build status](https://img.shields.io/travis/luobotang/simply-vue/master.svg?style=flat-square)](https://travis-ci.org/luobotang/simply-vue)

A simple VueJS.

## usage

```javascript
var vm = new Vue({
  el: '#box',
  data: {
    name: 'luobo'
  }
})
```
