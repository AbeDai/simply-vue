# simply-vue

[![Build Status](https://travis-ci.org/luobotang/simply-vue.svg?branch=master)](https://travis-ci.org/luobotang/simply-vue)

A simple VueJS.

## usage

```javascript
var vm = new Vue({
  el: '#box',
  data: {
    name: 'luobo'
  }
})
```
